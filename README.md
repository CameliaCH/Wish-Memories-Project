# Wish Memories: Photo Book Builder

A web-based tool built for Make-A-Wish to streamline the creation of personalised photo books for wish children following their Wish Day.

## What it does

The builder guides staff through a simple flow:

1. **Customise**: pick a colour scheme, font, shape style, and page design that suits the child, with a live preview of the result.
2. **Edit**: upload and arrange photos, add the child's name, age, and wish, and personalise pages with text and stickers.

Once finished, the tool exports a print-ready PDF that can be sent straight to a printing service, so the photo book can reach the child shortly after their Wish Day.

## Key features

- Theme system with customisable colours, fonts, and shape styles
- Drag-and-drop photo placement with reposition and zoom controls
- Custom text and sticker tools for personalisation
- Undo/redo history and autosave
- One-click export to a high-quality, print-ready PDF

## Tech

Built as a front-end web application using HTML, CSS, and JavaScript, with PDF generation handled via jsPDF and html2canvas.

## Status

The tool is largely complete and currently being refined alongside the Make-A-Wish team to improve quality and time efficiency.

# Stickers Folder

Place your sticker image files here (PNG, SVG, JPG, WebP — PNG with transparent background recommended).

Then open `js/config.js` and add the filenames to the `STICKER_FILES` array:

```js
const STICKER_FILES = [
  'my-sticker-1.png',
  'my-sticker-2.png',
  // etc.
];
```

The sticker grid in the editor sidebar will automatically display all listed files.

## Default placeholder stickers included:
- star.svg
- lightning.svg
- heart.svg
- shield.svg
- crown.svg
- pow.svg
- ribbon.svg
- wish-tag.svg

Replace or add to these as needed.
